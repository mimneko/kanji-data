# 人名漢字（子の名に使える漢字）[^4]
常用漢字 + 人名用漢字(常用漢字の異体字を含む) = 2999字(2769字種2999字体)

識別番号は独自に付けたもので、意味はない。

# 漢検級別漢字表（１０級～２級）
配当級は「日本漢字能力検定 級別漢字表（１０級～２級）」（２０２０年２月発表、公益財団法人 日本漢字能力検定協会発行）による。

ただし、「叱」に関しては表の字体から代わりに「𠮟」とした。

識別番号は人名漢字とリンクしており、同様に独自に付けたもので、意味はない。
